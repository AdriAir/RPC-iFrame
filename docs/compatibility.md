# Compatibility

## Browser Support

Requires modern browser APIs: `Proxy`, `Promise`, `postMessage`, `crypto.randomUUID()`.

| Browser       | Minimum Version |
| ------------- | --------------- |
| Chrome / Edge | 92+             |
| Firefox       | 95+             |
| Safari        | 15.4+           |
| Opera         | 78+             |

## Module Formats

- **ESM** — `import { connectIframe } from 'rpc-iframe'`
- **CJS** — `const { connectIframe } = require('rpc-iframe')`
- **TypeScript** — Full type definitions included
